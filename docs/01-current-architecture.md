## 1. Project Overview

### What It Is

A static single-page personal portfolio website for Abdallah Hassoun, a first-year CS student. The site showcases bio, skills, projects, and contact information with a clean, professional design.

### Tech Stack

| Layer               | Technology                  | Version/Notes                                         |
| ------------------- | --------------------------- | ----------------------------------------------------- |
| **Markup**          | HTML5                       | Semantic (nav, main, section, article, footer)        |
| **Styling**         | CSS3                        | 755 lines, single file, custom properties for theming |
| **Scripting**       | TypeScript вҶ’ JavaScript     | 147 lines source, compiled via`tsc`                   |
| **Icons**           | Font Awesome 6.5.1          | CDN-loaded (`css/all.min.css`)                        |
| **Fonts**           | Google Fonts                | Sora (headings) + DM Sans (body)                      |
| **Build**           | TypeScript Compiler (`tsc`) | No bundler, no transpiler pipeline                    |
| **Package Manager** | npm                         | Single dev dependency: TypeScript ^7.0.2              |
| **Deployment**      | Vercel                      | Auto-deploy from`main` branch                         |

### Deployment

- **vercel.json**: `{ "outputDirectory": "." }` вҖ” serves the root directory as-is
- No build command configured in Vercel (must run `npm run build` manually or script it)
- Vercel auto-detects static HTML output

### Scripts

```json
{
  "build": "tsc", // Compiles script.ts вҶ’ script.js
  "typecheck": "tsc --noEmit" // Type-check without emitting
}
```

## 9. Current Problems & Pain Points

### 9.1 Code Organization

| Issue                         | Detail                                                                                                                       |
| ----------------------------- | ---------------------------------------------------------------------------------------------------------------------------- |
| **Monolithic CSS**            | All 755 lines in one file with no separation of concerns. No CSS modules, no component-scoped styles.                        |
| **Single anonymous callback** | Entire`script.ts` is one anonymous DOMContentLoaded handler. Zero functions, zero modules, no separation of logic.           |
| **BEM inconsistency**         | Mix of BEM (`.navbar__links`) and kebab-case (`.project-card`, `.skills-list`, `.contact-links`). Naming convention drift.   |
