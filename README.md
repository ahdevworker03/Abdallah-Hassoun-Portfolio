## ✨ Features

- **Responsive Design:** Optimized for all screen sizes, from mobile to desktop.
- **Dark/Light Mode:** Seamless theme switching with persistent user preference using `localStorage`.
- **Scroll Animations:** Smooth entry animations for sections using the `Intersection Observer API`.
- **Clean UI:** Professional typography and spacing for a polished look.
- **Direct CV Download:** Easy access to my professional resume.

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3 (Custom Variables, Flexbox, Grid)
- **Scripting:** Vanilla JavaScript (ES6+)
- **Icons & Fonts:** Google Fonts (Sora, DM Sans)
- **Version Control:** Git & GitHub

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ahdevworker03/Abdallah-Hassoun-Portfolio.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd Abdallah-Hassoun-Portfolio
   ```
3. **Open `index.html`:**
   You can open the file directly in your browser or use an extension like "Live Server" in VS Code.

## 📜 Available Scripts

Since this is a static project, no build scripts are required. Simply serve the files using any static web server.

## 🌐 Deployment

The project is designed to be hosted on platforms like **GitHub Pages**, **Netlify**, or **Vercel**. To deploy on GitHub Pages:
1. Push your code to the `main` branch.
2. Go to **Settings > Pages** in your GitHub repository.
3. Select the `main` branch as the source and click Save.
